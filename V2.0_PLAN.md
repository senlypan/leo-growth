## 🎯 模块目标

帮助学生全面了解自己的学习能力，制定科学的学习路径和目标。

### 核心功能
1. **能力评估** - 语数英三科能力测评
2. **学习路径** - 知识点地图和学习顺序
3. **目标管理** - 学习目标制定和跟踪

## 🔧 技术设计

### 数据库设计

#### 评估表
```sql
-- 能力评估表
CREATE TABLE assessments (
    id BIGINT PRIMARY KEY,
    student_id BIGINT NOT NULL,
    subject VARCHAR(20) NOT NULL,
    assessment_type VARCHAR(50) NOT NULL,
    score INT NOT NULL,
    result_data JSON NOT NULL,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 评估题目表
CREATE TABLE assessment_questions (
    id BIGINT PRIMARY KEY,
    subject VARCHAR(20) NOT NULL,
    question_type VARCHAR(50) NOT NULL,
    question_content TEXT NOT NULL,
    options JSON,
    correct_answer VARCHAR(500) NOT NULL,
    score INT NOT NULL
);
```

#### 知识点表
```sql
-- 知识点地图表
CREATE TABLE knowledge_points (
    id BIGINT PRIMARY KEY,
    subject VARCHAR(20) NOT NULL,
    grade INT NOT NULL,
    point_name VARCHAR(200) NOT NULL,
    parent_id BIGINT,
    sort_order INT,
    description TEXT
);

-- 学生知识点掌握表
CREATE TABLE student_knowledge (
    id BIGINT PRIMARY KEY,
    student_id BIGINT NOT NULL,
    point_id BIGINT NOT NULL,
    mastery_level VARCHAR(20) NOT NULL,
    learned_at DATETIME,
    reviewed_at DATETIME
);
```

#### 目标表
```sql
-- 学习目标表
CREATE TABLE learning_goals (
    id BIGINT PRIMARY KEY,
    student_id BIGINT NOT NULL,
    goal_type VARCHAR(20) NOT NULL,
    goal_title VARCHAR(200) NOT NULL,
    goal_description TEXT,
    target_date DATE,
    progress INT DEFAULT 0,
    status VARCHAR(20) DEFAULT 'active',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

### API 设计

#### 能力评估 API
```
POST   /assessment/start         # 开始评估
GET    /assessment/questions     # 获取题目
POST   /assessment/submit        # 提交答案
GET    /assessment/report        # 评估报告
GET    /assessment/history       # 历史记录
```

#### 学习路径 API
```
GET    /learning/path            # 获取学习路径
GET    /learning/knowledge-map   # 知识点地图
POST   /learning/mark-learned    # 标记已掌握
GET    /learning/next-step       # 下一步推荐
```

#### 目标管理 API
```
GET    /goals                    # 目标列表
POST   /goals                    # 创建目标
PUT    /goals/{id}               # 更新目标
PUT    /goals/{id}/progress      # 更新进度
DELETE /goals/{id}               # 删除目标
```
