# 动画和庆祝效果设计

> ✨ 让每次成功都值得庆祝！  
> **适用年龄:** 7-12 岁小学生  
> **设计原则:** 即时、夸张、有趣

---

## 🎆 一、成功庆祝动画

### 1.1 作业完成庆祝

#### 基础效果
```
触发条件：完成一次作业
动画内容:
  ✅ 绿色大对勾从中心弹出（放大 + 旋转）
  🌟 星星从四周飞向中心
  🎉 彩带从顶部飘落
  📊 积分数字滚动 +10

持续时间：2 秒
音效：欢快的"叮叮叮"
文案："太棒了！又完成了一次作业！"
```

#### 全对额外庆祝
```
触发条件：作业全对
动画内容:
  🌟 金色星星闪烁（更大更亮）
  🎊 烟花绽放效果
  🏆 奖杯从底部升起
  📊 积分数字滚动 +15（10+5 额外）

持续时间：3 秒
音效：更欢快的"啦啦啦～"
文案："哇！全对！你太厉害了！"
```

## ✨ 二、日常交互动画

### 2.1 按钮点击反馈

#### 主要按钮
```
按下时:
  - 缩放至 0.95 倍（100ms）
  - 颜色变深
  - 轻微阴影

松开时:
  - 恢复原状（100ms）
  - 颜色恢复
  - 阴影恢复

触感：轻微震动（如果支持）
```

#### 次要按钮
```
按下时:
  - 缩放至 0.98 倍（100ms）
  - 背景色变深

松开时:
  - 恢复原状（100ms）

触感：无
```

### 2.2 卡片悬停效果

```
鼠标悬停/手指长按:
  - 上浮 4rpx（200ms）
  - 阴影加深
  - 轻微放大（1.02 倍）

离开时:
  - 恢复原状（200ms）
  - 阴影恢复
```

### 2.3 进度条动画

#### 增长动画
```
触发：进度增加
动画:
  - 进度条从左到右填充（0.3s）
  - 颜色渐变（橙→绿）
  - 数字滚动效果

 easing: ease-out
```

#### 完成动画
```
触发：进度达到 100%
动画:
  - 进度条闪烁 3 次（0.5s×3）
  - 颜色变金色
  - 小星星从进度条飞出

 easing: ease-in-out
```

## 🎯 六、动画性能优化

### 6.1 性能要求

```
帧率：≥60fps
加载时间：<100ms
持续时间：1-5 秒
CPU 占用：<10%
内存占用：<50MB
```

### 6.2 优化策略

```
1. 使用 CSS transform 代替位置变化
2. 使用 requestAnimationFrame
3. 避免重排重绘
4. 使用硬件加速
5. 控制同时播放的动画数量
6. 低性能设备简化动画
```

### 6.3 降级方案

```
低端设备:
  - 减少粒子数量
  - 简化动画效果
  - 缩短动画时长
  - 禁用复杂特效

用户设置:
  - 提供"减少动画"选项
  - 提供"关闭动画"选项
  - 保留必要反馈动画
```
