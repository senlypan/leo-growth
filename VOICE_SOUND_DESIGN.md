## 🎯 四、语音录制规范

### 4.1 声音要求

#### 声音类型
```
主要：温暖的女声（适合小学生）
备选：温暖的男声
风格：亲切、友好、鼓励
年龄感：25-35 岁
```

#### 录音质量
```
格式：WAV/MP3
采样率：44.1kHz
位深度：16bit
比特率：128kbps 以上
```

### 4.2 录制环境

```
环境：专业录音棚
噪音：<30dB
混响： minimal
回声：无
```

### 4.3 情感表达

```
鼓励：温暖、真诚
赞美：兴奋、真实
关心：温和、真诚
庆祝：欢快、自然
```

## 📱 五、技术实现

### 5.1 播放策略

#### 预加载
```
常用音效：启动时预加载
语音提示：按需加载
背景音乐：按需加载
```

#### 并发控制
```
最多同时播放：3 个音效
语音优先级：最高
音效优先级：中等
音乐优先级：最低
```

#### 中断处理
```
新语音打断旧语音：是
新音效叠加旧音效：是（最多 3 个）
音乐可被语音降低音量：是
```

### 5.2 性能优化

```
音效压缩：是
按需加载：是
缓存策略：常用音效缓存
内存限制：<50MB
```

### 5.3 兼容性

```
iOS: 支持
Android: 支持
微信：支持
低性能设备：自动降级
```
