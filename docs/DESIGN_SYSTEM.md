# Leo Growth 设计系统

> **Think Different** - 像乔布斯一样思考  
> **设计哲学:** 极简、优雅、直观、情感化  
> **目标用户:** 6-12 岁儿童  
> **版本:** v2.0 (乔布斯设计版)

## 🎯 设计原则

### 1. 极简主义 (Simplicity)
```
少即是多 (Less is More)
没有多余元素
每个元素都有意义
```

### 2. 优雅精致 (Elegance)
```
渐变色彩
柔和阴影
微妙动画
完美细节
```

### 3. 直观易用 (Intuitive)
```
大字体 - 易读
大图标 - 易识别
大间距 - 易点击
一目了然
```

### 4. 情感连接 (Emotional)
```
温暖色彩
友好 emoji
鼓励话语
有趣互动
```

## ✨ 动画系统

### 缓动函数
```css
cubic-bezier(0.4, 0, 0.2, 1) /* 标准 */
cubic-bezier(0.25, 0.1, 0.25, 1) /* 平滑 */
cubic-bezier(0.68, -0.55, 0.265, 1.55) /* 弹性 */
```

### 动画时长
```
快速：0.2s - 按钮反馈
标准：0.3s - 卡片动画
慢速：0.5s - 页面切换
极慢：0.6s - 入场动画
```

### 关键帧动画
```css
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(40rpx); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes shimmer {
  0%, 100% { transform: translate(0, 0); }
  50% { transform: translate(-20%, -20%); }
}

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-16rpx); }
}

@keyframes glow {
  0%, 100% { box-shadow: 0 0 40rpx rgba(102, 126, 234, 0.3); }
  50% { box-shadow: 0 0 60rpx rgba(102, 126, 234, 0.5); }
}
```

## 🎨 组件设计

### 1. 卡片 (Card)
```
特征:
- 大圆角 (24-28rpx)
- 柔和阴影
- 白色背景
- 充足内边距 (32rpx)
- 顶部彩色条 (6rpx)

用途:
- 内容展示
- 功能入口
- 数据统计
```

### 2. 按钮 (Button)
```
特征:
- 大圆角 (20-36rpx)
- 渐变背景
- 大字体 (30rpx)
- 点击缩放 (0.95)
- 阴影光晕

类型:
- 主要按钮 (紫色渐变)
- 成功按钮 (绿色渐变)
- 警告按钮 (橙色渐变)
- 次要按钮 (描边)
```

### 3. 胶囊 (Capsule)
```
特征:
- 椭圆形状
- 小尺寸
- 选中渐变
- 点击缩放

用途:
- 筛选器
- 标签
- 状态标识
```

### 4. 徽章 (Badge)
```
特征:
- 小圆角
- 浅色背景
- 彩色文字
- emoji 图标

用途:
- 状态标识
- 角标提示
- 分类标签
```
