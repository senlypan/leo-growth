# 积分和成就系统设计

> 🏆 让学习更有趣，更有成就感  
> **适用年龄:** 7-12 岁小学生

---

## 🌟 积分系统

### 获取积分

#### 日常任务
```
✅ 完成作业：+10 分/次
✅ 作业全对：+5 分额外奖励
✅ 错题复习：+5 分/题
✅ 每日登录：+3 分
✅ 连续打卡：+3 分/天（翻倍递增）
```

#### 评估测试
```
📊 完成评估：+20 分/次
📊 评估 90 分以上：+10 分额外奖励
📊 评估进步 10 分：+15 分奖励
```

#### 目标达成
```
🎯 完成短期目标：+30 分
🎯 完成长期目标：+100 分
🎯 连续完成目标：+20 分额外奖励
```

### 积分展示

#### 首页展示
```
┌─────────────────────────────────┐
│  🌟 我的积分：1,250             │
│  🔥 连续打卡：7 天               │
├─────────────────────────────────┤
│  今日获得：+23 分               │
│  - 完成作业：+10 分              │
│  - 作业全对：+5 分               │
│  - 错题复习：+5 分×2=10 分        │
│  - 每日登录：+3 分               │
└─────────────────────────────────┘
```

#### 积分明细
```
┌─────────────────────────────────┐
│  📅 2026-03-12                  │
│  +10 完成数学作业               │
│  +5  作业全对奖励               │
│  +5  复习错题：两位数加法        │
│  +5  复习错题：乘法口诀          │
│  +3  每日登录奖励               │
├─────────────────────────────────┤
│  今日合计：+28 分               │
└─────────────────────────────────┘
```

### 积分用途

#### 虚拟奖励
```
🎨 主题皮肤：50-200 分
🎭 头像框：100-300 分
🏷️ 专属徽章：200-500 分
✨ 特效动画：150-400 分
```

#### 实物奖励（需家长确认）
```
📚 课外书：500-1000 分
✏️ 文具套装：800-1500 分
🎮 游戏时间：1000-2000 分
🎁 神秘礼物：2000-5000 分
```

## 🎁 奖励机制

### 即时奖励

#### 完成作业
```
🎉 弹窗庆祝："太棒了！又完成了一次作业！"
✨ 动画效果：彩带飘落
🌟 积分奖励：+10 分
📊 进度更新：作业完成 +1
```

#### 作业全对
```
🎉 特殊弹窗："哇！全对！你太厉害了！"
✨ 特殊动画：星星闪烁
🌟 额外奖励：+5 分
🏆 成就进度：正确率大师 +1
```

#### 连续打卡
```
🔥 火焰动画："已经坚持 X 天了！"
🌟 奖励翻倍：第 N 天获得 N×3 分
🏆 成就解锁：连续打卡达人
```

### 阶段性奖励

#### 每日奖励
```
📅 完成今日所有任务
🎁 奖励：+20 分
🏆 成就：今日事今日毕
```

#### 每周奖励
```
📅 连续 7 天打卡
🎁 奖励：+100 分
🏆 成就：周冠军
🎨 解锁：专属头像框（7 天）
```

#### 每月奖励
```
📅 连续 30 天打卡
🎁 奖励：+500 分
🏆 成就：月冠军
🎨 解锁：专属头像框（30 天）
🎁 实物奖励：文具套装（需家长确认）
```
