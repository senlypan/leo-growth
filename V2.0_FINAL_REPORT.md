## 📈 技术架构

### 后端架构
```
Controller Layer (RESTful API)
    ↓
Service Layer (业务逻辑)
    ↓
Mapper Layer (数据访问)
    ↓
Database (MySQL)
```

### 前端架构
```
Pages (页面层)
    ↓
Components (组件层)
    ↓
Utils (工具层)
    ↓
API (数据层)
```

### 数据流
```
用户操作 → 前端页面 → API 请求 → 后端 Controller
    ↓
后端 Service → 后端 Mapper → 数据库
    ↓
数据库 → 后端 Mapper → 后端 Service → 后端 Controller
    ↓
API 响应 → 前端页面 → UI 更新
```
