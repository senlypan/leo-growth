# Leo Growth 迭代开发计划

> 持续迭代，直到完成所有功能

## 🎯 最终目标

在一个微信小程序中实现：
1. **作业辅导员** 📚 - 替代父母辅导孩子做作业
2. **学业规划师** 📖 - 规划语文、数学、英语的学习路径
3. **体能规划师** 🏃 - 根据体能成绩做整体规划

## 📋 迭代策略

- **小步快跑** - 每个版本完成一个核心功能
- **快速迭代** - 完成一个功能立即测试
- **用户导向** - 以潘灏成（二年级）的实际需求为准
- **质量优先** - 每个功能都要完整可用

## 🔧 开发规范

### 代码规范
- 使用 ESLint 检查
- 中文注释
- 函数不超过 50 行
- 组件化开发

### 提交规范
```
feat: 新增功能
fix: 修复 bug
docs: 文档更新
style: 代码格式
refactor: 重构
test: 测试
chore: 构建/工具
```

### 测试规范
- 每个页面都要真机测试
- 关键功能要多次测试
- 发现 bug 立即修复
