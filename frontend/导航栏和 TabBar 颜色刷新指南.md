# 🔄 导航栏和 TabBar 颜色刷新指南

## 🎯 当前配置

### app.json 配置
```json
{
  "window": {
    "navigationBarBackgroundColor": "#07C160",  // ✅ 导航栏背景：微信绿
    "navigationBarTitleText": "Leo Growth",
    "navigationBarTextStyle": "white"           // ✅ 标题文字：白色
  },
  "tabBar": {
    "color": "#999999",                         // ✅ 默认文字：灰色
    "selectedColor": "#07C160",                 // ✅ 选中文字：微信绿
    "backgroundColor": "#ffffff",               // ✅ 背景：白色
    "list": [...]
  }
}
```

---

## ❌ 为什么颜色没变？

### 可能原因
1. **微信开发者工具缓存** - 最常见
2. **app.json 文件未保存**
3. **需要完全重启微信开发者工具**
4. **基础库版本问题**

---

## ✅ 解决方案

### 方案 1: 清除缓存 + 重新编译
```
1. 微信开发者工具 → 工具 → 清除缓存 → 清除全部
2. 点击编译按钮
3. 查看效果
```

### 方案 2: 重启微信开发者工具
```
1. 关闭微信开发者工具
2. 重新打开项目
3. 编译查看
```

### 方案 3: 修改基础库版本
```
在 app.json 中添加：
{
  "style": "v2",
  "sitemapLocation": "sitemap.json",
  "lazyCodeLoading": "requiredComponents"
}
```

## 🎨 应该显示的颜色

### 导航栏（页面顶部）
- **背景**：🟢 微信绿 `#07C160`
- **标题文字**：⚪ 白色 `#FFFFFF`
- **标题内容**："Leo Growth" 或页面标题

### TabBar（页面底部）
- **背景**：⚪ 白色 `#FFFFFF`
- **默认文字**：⚫ 灰色 `#999999`
- **选中文字**：🟢 微信绿 `#07C160`
- **文字内容**：首页、作业、错题、报告

---

## 📸 检查点

### 导航栏
- [ ] 背景是绿色不是蓝色/紫色
- [ ] "Leo Growth" 文字是白色
- [ ] 返回按钮是白色（如果有）

### TabBar
- [ ] 底部背景是白色
- [ ] 未选中的文字是灰色
- [ ] 选中的文字是绿色
- [ ] 选中时图标/文字颜色一致

---

## 🆘 如果还是不对

### 检查 app.json 是否保存
```
1. 打开 app.json
2. 随便加个空格再删除（触发保存）
3. Ctrl + S 保存
4. 重新编译
```

### 检查是否有页面级覆盖
```
查看各个页面的 .json 文件
确保没有 navigationBarBackgroundColor 配置
```
