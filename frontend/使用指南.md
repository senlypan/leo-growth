# Leo Growth 使用指南

## 🎯 快速开始

### 第一次使用

1. **打开小程序**
   - 首次进入会看到空状态
   - 这是正常的，因为还没有数据

2. **录入第一道作业**
   ```
   首页 → 作业辅导 → + 录入作业
   ```
   - 选择科目（语文/数学/英语）
   - 拍照上传作业
   - AI 自动识别题目
   - 确认内容并提交

3. **完成作业**
   ```
   作业列表 → 点击作业 → 完成作业
   ```
   - 拍照上传完成的作业
   - 提交给家长批改

4. **家长批改**
   ```
   作业详情 → 家长批改
   ```
   - AI 辅助批改
   - 写评语
   - 提交

## 📚 错题本使用流程

### 录入错题的三种方式

#### 方式 1：手动录入
```
错题本 → 录入错题
```
- 选择科目
- 输入题目内容
- 输入你的答案
- 输入正确答案
- 选择错误类型
- 填写知识点
- 提交

#### 方式 2：从作业加入
```
作业详情 → 加入错题本
```
- 自动带入职题目内容
- 只需补充答案和错因

#### 方式 3：拍照识别
```
错题本 → 录入错题 → 拍照
```
- 拍照上传错题
- AI 识别题目
- 补充答案即可

### 复习错题

```
错题本 → 智能复习
```

1. 系统根据艾宾浩斯曲线推荐
2. 重做题目
3. 对比答案
4. 自我评估
5. 安排下次复习

## 💡 使用技巧

### 1. 拍照技巧
- 光线充足
- 纸张平整
- 对焦清晰
- 完整拍摄

### 2. AI 识别
- 识别后请检查
- 可以手动修改
- 支持多语言

### 3. 错题分类
- 计算错误：粗心算错
- 概念不清：知识点不懂
- 审题错误：理解题意有误
- 方法不当：解题方法不对

### 4. 复习计划
- 每天复习待复习的错题
- 标记已掌握的错题
- 系统会自动安排复习时间

## ❓ 常见问题

### Q: 为什么显示空状态？
A: 第一次使用是正常的，开始录入作业或错题即可。

### Q: AI 识别不准确怎么办？
A: 可以手动修改识别结果，确保内容正确。

### Q: 如何删除作业/错题？
A: 在详情页面向左滑动，点击删除按钮。

### Q: 可以导出错题吗？
A: 可以，点击"导出错题"生成 PDF。

### Q: 如何邀请家长批改？
A: 完成作业后提交，家长会在微信收到通知。
