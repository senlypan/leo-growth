## 🎨 统一后的设计

### 颜色
```
主色：#07C160（微信绿）
背景：#F5F5F5（浅灰）+ #FFFFFF（纯白）
文字：#111111 / #666666 / #999999
```

### Class 命名规范
```
旧：welcome-title-gradient、stat-value-gradient、premium-card
新：welcome-title、stat-value、welcome-card
```

### 设计原则
- 简洁：去除所有"premium"、"gradient"等复杂命名
- 统一：所有 class 名称简洁明了
- 克制：不再使用花哨的视觉效果
