## 🔧 需要开发的后端功能

### 1. AI 识别
```javascript
// OCR 识别
POST /api/ocr/recognize
Body: { image: base64 }
Response: { text: string, confidence: number }

// 题目分类
POST /api/ocr/classify
Body: { text: string }
Response: { subject: string, type: string, knowledgePoints: [] }
```

### 2. 作业管理
```javascript
// 创建作业
POST /api/homework
Body: { subject, content, images, deadline }

// 提交作业
POST /api/homework/:id/submit
Body: { images, note }

// 批改作业
POST /api/homework/:id/check
Body: { isCorrect, score, comment }
```

### 3. 错题管理
```javascript
// 创建错题
POST /api/mistakes
Body: { subject, question, userAnswer, correctAnswer, errorType }

// 复习错题
POST /api/mistakes/:id/review
Body: { newAnswer, isMastered }
```

### 4. 积分系统
```javascript
// 获得积分
POST /api/points/earn
Body: { action: string, points: number }

// 积分记录
GET /api/points/history
```
