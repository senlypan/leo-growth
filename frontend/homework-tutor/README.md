# Homework Tutor - 作业辅导员

微信小程序前端模块，替代父母辅导孩子做作业。

## 核心功能

### 1. 作业管理
- 查看每日作业（语文、数学、英语）
- 作业状态跟踪（未完成/已完成/已检查）
- 作业提醒

### 2. 智能辅导
- AI 题目讲解
- 拍照搜题
- 知识点关联

### 3. 作业检查
- 自动批改
- 错题标注
- 检查报告

### 4. 错题本
- 自动收录错题
- 错题分类
- 智能复习计划

### 5. 学情分析
- 学习报告
- 进步趋势
- 薄弱点识别

## 页面说明

### index (首页)
- 展示当日作业列表
- 作业状态筛选
- 快速入口（错题本、学习报告）

### homework (作业详情)
- 作业详细内容
- 提交入口
- 历史提交记录

### submit (提交作业)
- 拍照上传
- 文字输入
- 提交确认

### check (作业检查)
- AI 自动批改结果
- 错题标注
- 检查报告

### mistakes (错题本)
- 错题列表
- 错题分类筛选
- 错题详情
- 复习打卡

### report (学习报告)
- 学习数据统计
- 进步趋势图
- 薄弱点分析

### profile (个人中心)
- 用户信息
- 学习统计
- 设置
