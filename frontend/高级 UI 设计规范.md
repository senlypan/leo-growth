# Leo Growth - 高级 UI 设计规范

## 🎯 设计理念

### 核心原则
1. **少即是多** - 去除一切不必要的元素
2. **克制** - 颜色、动画、装饰都要克制
3. **精致** - 细节决定品质
4. **留白** - 空间是高级感的关键
5. **统一** - 一致的视觉语言

---

## 🎨 颜色系统

### 主色（单一克制）
```
#07C160 - 微信绿（唯一主色）
```

### 中性色（精致层次）
```
#1A1A1A - 主文字
#666666 - 次文字
#999999 - 辅助文字
#CCCCCC - 占位符
#E8EAED - 边框
#F0F2F5 - 浅背景
#F7F8FA - 页面背景
#FFFFFF - 卡片背景
```

### ❌ 禁止使用
- 花哨的渐变（除主色轻微渐变）
- 高饱和度颜色
- 五颜六色的 emoji
- 彩虹色效果

## 📐 间距系统

### 严格 8 倍数
```
8rpx   - 最小间距
16rpx  - 标准间距
24rpx  - 卡片内边距
32rpx  - 大间距
48rpx  - 超大间距
```

---

## 🔤 字体排印

### 字号层级
```
40rpx - 页面标题（字重 600）
32rpx - 章节标题（字重 600）
28rpx - 正文（字重 400/500）
26rpx - 次要文字（字重 400）
24rpx - 辅助文字（字重 400）
```

### 字间距
```
标题：-0.5rpx（精致紧凑）
正文：0（正常）
```

## 🎴 卡片设计

### 标准卡片
```css
.card {
  background: #FFFFFF;
  border-radius: 16rpx;
  padding: 32rpx;
  box-shadow: 0 2rpx 8rpx rgba(0,0,0,0.04);
}
```

### 设计要点
- 统一的圆角
- 极其轻微的阴影
- 足够的内边距
- 干净的边框

---

## 🚫 去除 emoji

### 为什么不用 emoji
1. **不专业** - emoji 是 casual 的，不是 professional 的
2. **不统一** - 每个 emoji 风格不同
3. **不精致** - 细节粗糙
4. **不高级** - 像儿童应用

### 替代方案
- **纯色块图标** - 简洁几何图形
- **文字** - 直接表达
- **极简线条图标** - 统一风格

### 对比
```
❌ 之前：📚 作业辅导、📖 学业规划、🏃 体能训练
✅ 现在：作业辅导、学业规划、体能训练（配纯色块）
```

## 🎬 动画原则

### 克制使用
- 只在必要时使用
- 时间 0.15-0.3s
- 缓动函数统一
- 不炫技

### 标准动画
```css
/* 按压反馈 */
.btn:active {
  opacity: 0.7;
  transform: scale(0.98);
}

/* 淡入 */
.fade-in {
  animation: fadeIn 0.2s ease;
}
```

## 📊 高级感的关键

### 1. 留白
- 元素之间足够的空间
- 不拥挤
- 呼吸感

### 2. 统一
- 圆角统一
- 间距统一
- 颜色统一
- 字重统一

### 3. 层次
- 文字三级颜色
- 阴影轻微
- 边框细腻

### 4. 克制
- 不用多余装饰
- 不用花哨效果
- 不用复杂渐变

### 5. 精致
- 字间距微调
- 阴影精确控制
- 边框细腻
