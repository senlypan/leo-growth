# Study Planner - 学业规划师

微信小程序前端模块，为潘灏成规划语文、数学、英语的学习路径。

## 核心功能

### 1. 学科评估
- 语文能力评估
- 数学能力评估
- 英语能力评估
- 综合分析报告

### 2. 学习路径
- 个性化学习路径规划
- 知识点地图
- 学习顺序建议

### 3. 目标管理
- 短期目标（周/月）
- 长期目标（学期/学年）
- 目标进度跟踪

### 4. 进度跟踪
- 学习进度可视化
- 完成情况统计
- 时间投入分析

### 5. 学习建议
- 薄弱点分析
- 个性化建议
- 学习资源推荐
