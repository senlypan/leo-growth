# Leo Growth - 完整功能闭环设计

## 🎯 核心功能流程

### 1. 作业辅导 - 完整闭环

```
┌─────────────────────────────────────────────────────────────┐
│                    作业辅导完整流程                          │
└─────────────────────────────────────────────────────────────┘

1️⃣ 作业列表页 (homework.wxml)
   └─ 查看今日作业
   └─ 筛选（科目/状态）
   └─ 点击「+ 录入作业」→ 2️⃣

2️⃣ 作业录入页 (homework/add/add.wxml) 【新增】
   └─ 选择科目（语文/数学/英语）
   └─ 拍照上传（支持多图）
   └─ AI 识别题目内容
   └─ 手动补充/修正
   └─ 设置截止时间
   └─ 提交 → 3️⃣

3️⃣ AI 识别处理中 (homework/recognizing.wxml) 【新增】
   └─ 显示识别进度
   └─ 识别题目文字
   └─ 分类题目类型
   └─ 识别完成 → 4️⃣

4️⃣ 作业详情页 (homework/detail/detail.wxml)
   └─ 查看作业内容
   └─ 查看 AI 识别结果
   └─ 点击「完成作业」→ 5️⃣

5️⃣ 提交作业页 (homework/submit/submit.wxml)
   └─ 拍照上传完成后的作业
   └─ 填写完成心得
   └─ 提交 → 6️⃣

6️⃣ 家长批改页 (homework/check/check.wxml)
   └─ 查看孩子提交的作业
   └─ AI 辅助批改（识别正确率）
   └─ 家长确认/修改
   └─ 写评语
   └─ 提交批改 → 7️⃣

7️⃣ 作业完成页 (homework/completed.wxml) 【新增】
   └─ 显示批改结果
   └─ 显示评语
   └─ 获得积分/奖励
   └─ 查看错题解析
   └─ 加入错题本（可选）
   └─ 完成闭环 ✅
```

### 2. 错题本 - 完整闭环

```
┌─────────────────────────────────────────────────────────────┐
│                     错题本完整流程                           │
└─────────────────────────────────────────────────────────────┘

1️⃣ 错题列表页 (mistakes.wxml)
   └─ 查看错题
   └─ 筛选（科目/状态）
   └─ 点击「录入错题」→ 2️⃣

2️⃣ 错题录入页 (mistakes/submit/submit.wxml)
   └─ 选择科目
   └─ 拍照/上传错题
   └─ AI 识别题目
   └─ 填写错误答案
   └─ 填写正确答案
   └─ 选择错误类型（计算错误/概念不清/审题错误）
   └─ 关联知识点
   └─ 提交 → 3️⃣

3️⃣ 错题详情页 (mistakes/detail/detail.wxml)
   └─ 查看题目
   └─ 查看错因分析
   └─ 查看正确答案
   └─ 查看知识点讲解
   └─ 点击「复习」→ 4️⃣

4️⃣ 错题复习页 (mistakes/review/review.wxml) 【新增】
   └─ 重做题目
   └─ 提交答案
   └─ 对比正确答案
   └─ 自我评估（掌握/未掌握）
   └─ 标记为已掌握 → 5️⃣

5️⃣ 复习完成页 (mistakes/reviewed.wxml) 【新增】
   └─ 显示复习结果
   └─ 显示掌握程度
   └─ 智能推荐下次复习时间
   └─ 获得复习积分
   └─ 完成闭环 ✅
```

```

   └─ 推荐练习题
   └─ 制定改进计划
   └─ 保存计划 → 4️⃣

4️⃣ 计划执行追踪 (report/plan/plan.wxml) 【新增】

   └─ 每日打卡
   └─ 进度追踪
   └─ 完成度统计
   └─ 完成闭环 ✅
```

## 🎯 每个功能必须有

### 1. 入口
- 清晰的入口
- 明确的说明

### 2. 操作
- 简单的操作流程
- 智能辅助（AI 识别等）

### 3. 反馈
- 实时进度反馈
- 操作结果反馈

### 4. 结果
- 清晰的完成状态
- 数据展示

### 5. 激励
- 积分/奖励
- 成就系统

### 6. 下一步
- 智能推荐
- 引导继续

---

## 🔄 闭环原则

```
开始 → 操作 → 反馈 → 结果 → 激励 → 下一步 → 循环
```

**每个功能都要能走通，有始有终！**
