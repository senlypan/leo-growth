# Fitness Planner - 体能规划师

微信小程序前端模块，根据孩子体能成绩做整体规划。

## 核心功能

### 1. 体测记录
- 身高体重记录
- 肺活量测试
- 50 米跑成绩
- 坐位体前屈
- 1 分钟跳绳
- 1 分钟仰卧起坐

### 2. 运动计划
- 个性化运动计划
- 每日训练任务
- 计划难度调整

### 3. 训练打卡
- 每日训练记录
- 打卡日历
- 连续打卡统计

### 4. 数据统计
- 成绩趋势图
- 对比分析
- 达标情况

### 5. 健康建议
- 运动建议
- 营养建议
- 休息建议

## 训练维度

### 耐力训练
- 跑步练习
- 游泳
- 骑自行车

### 力量训练
- 仰卧起坐
- 俯卧撑
- 深蹲

### 柔韧性训练
- 坐位体前屈
- 拉伸运动
- 瑜伽基础

### 协调性训练
- 跳绳
- 平衡练习
- 球类运动
