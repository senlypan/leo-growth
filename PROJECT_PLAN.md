# Leo Growth 项目规划

> 潘灏成成长规划系统 - 微信小程序

## 📋 项目概述

**项目名称:** Leo Growth（灏成成长）  
**目标用户:** 潘灏成（二年级）  
**家长:** 小主人  
**项目类型:** 微信小程序 + Java SpringBoot 后端  

## 🎯 核心目标

1. **作业辅导员** - 替代父母辅导孩子做作业
2. **学业规划师** - 规划语文、数学、英语的学习路径
3. **体能规划师** - 根据体能成绩做整体规划

## 🎨 功能模块详细设计

### 1. 作业辅导员 (Homework Tutor) 📚

**核心功能:**
- 查看每日作业（语文、数学、英语）
- AI 智能辅导（题目讲解）
- 拍照搜题
- 作业自动检查
- 错题本管理
- 学习报告生成

**页面设计:**
```
首页 (index)
├── 今日作业概览
├── 作业完成状态
└── 快捷入口（错题本、学习报告）

作业列表 (homework)
├── 按日期筛选
├── 按科目筛选
├── 按状态筛选（未完成/已完成/已检查）
└── 作业卡片（科目、内容、截止时间）

作业详情 (detail)
├── 作业详细内容
├── 提交记录
├── 检查结果
└── AI 讲解入口

提交作业 (submit)
├── 拍照上传
├── 文字输入
└── 提交确认

错题本 (mistakes)
├── 错题列表（按科目分类）
├── 错题详情（题目、答案、解析）
├── 错题复习打卡
└── 错题导出

学习报告 (report)
├── 学习数据统计
├── 正确率趋势图
├── 薄弱点分析
└── 学习建议
```

**技术实现:**
- 微信小程序原生开发
- 图片上传（wx.chooseImage + wx.uploadFile）
- AI 接口调用（题目解答、作业检查）
- 图表展示（使用小程序图表库）

---

### 2. 学业规划师 (Study Planner) 📖

**核心功能:**
- 学科能力评估
- 学习路径规划
- 目标设定与跟踪
- 学习进度可视化
- 个性化学习建议

**页面设计:**
```
能力评估 (assessment)
├── 语文评估（识字、阅读、写作）
├── 数学评估（计算、几何、应用题）
├── 英语评估（词汇、听说、阅读）
└── 评估报告

学习路径 (path)
├── 知识点地图
├── 学习顺序建议
├── 已完成标记
└── 下一步推荐

目标管理 (goals)
├── 短期目标（周/月）
├── 长期目标（学期/学年）
├── 目标进度跟踪
└── 目标完成庆祝

进度跟踪 (progress)
├── 学习时长统计
├── 知识点掌握度
├── 进步曲线图
└── 学习日历
```

**技术实现:**
- 评估问卷系统
- 知识点图谱展示
- 进度可视化（图表）
- 目标提醒功能

---

### 3. 体能规划师 (Fitness Planner) 🏃

**核心功能:**
- 体能测试记录
- 运动计划制定
- 训练打卡
- 数据分析
- 健康建议

**页面设计:**
```
体测记录 (test)
├── 身高体重
├── 肺活量
├── 50 米跑
├── 坐位体前屈
├── 1 分钟跳绳
├── 1 分钟仰卧起坐
└── 体测历史

运动计划 (plan)
├── 每日训练任务
├── 周计划展示
├── 计划难度调整
└── 训练视频/图解

训练打卡 (training)
├── 打卡日历
├── 完成记录
├── 连续打卡统计
└── 打卡分享

数据统计 (stats)
├── 体能趋势图
├── 对比分析（与同龄人）
├── 达标情况
└── 健康建议
```

**技术实现:**
- 体测数据录入
- 打卡日历组件
- 数据可视化
- 健康算法（BMI、体能评分）

## 🔧 技术架构

### 前端技术栈
- **框架:** 微信小程序原生
- **UI 组件:** Vant Weapp / TDesign
- **图表:** F2 / ECharts for Weixin
- **状态管理:** 小程序全局数据 + Storage
- **网络请求:** wx.request 封装

### 后端技术栈
- **框架:** Spring Boot 3.2.0
- **数据库:** MySQL 8.0+
- **ORM:** MyBatis-Plus
- **认证:** JWT
- **文件存储:** 本地存储 / 云存储

### 数据库设计

**核心表:**
1. `users` - 用户表
2. `homeworks` - 作业表
3. `homework_submissions` - 作业提交表
4. `mistakes` - 错题表
5. `learning_reports` - 学习报告表
6. `study_plans` - 学习计划表
7. `fitness_records` - 体能记录表
8. `fitness_plans` - 体能计划表

## 📝 开发规范

### 代码规范
- 使用 ESLint 检查代码
- 遵循微信小程序开发规范
- 注释完整（中文注释）

### 命名规范
- 文件名：kebab-case（如：homework-list.js）
- 组件名：PascalCase（如：HomeworkCard）
- 变量名：camelCase（如：studentName）
- 常量名：UPPER_SNAKE_CASE（如：API_BASE_URL）

### Git 规范
- 分支管理：main / dev / feature/*
- 提交信息：feat: 新增功能 / fix: 修复 bug / docs: 文档更新
- 代码审查：重要功能需要审查
