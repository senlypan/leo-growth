# v1.4 学习报告功能 - 开发计划

> 📊 学习数据统计与分析  
> **预计开始:** 2026-03-12  
> **版本:** v1.4

---

## 🎯 功能目标

在一个页面中展示学生的学习全景：
1. **学习概览** - 核心数据一目了然
2. **作业统计** - 完成情况/正确率趋势
3. **错题分析** - 科目分布/知识点薄弱点
4. **学习建议** - AI 生成的个性化建议

---

## 📋 功能清单

### 前端页面
- [ ] **学习报告主页** (`pages/report/report.*`)
  - 学习概览卡片（总览数据）
  - 作业完成统计（柱状图）
  - 正确率趋势（折线图）
  - 错题科目分布（饼图）
  - 薄弱知识点 TOP5
  - 学习建议列表

- [ ] **详细报告页** (`pages/report/detail/detail.*`)
  - 日报/周报/月报切换
  - 详细数据统计
  - 历史对比
  - 导出/分享功能

### 后端 API
- [ ] **ReportController**
  - `GET /report/overview?studentId=&period=` - 学习概览
  - `GET /report/homework-stats?studentId=&period=` - 作业统计
  - `GET /report/mistake-analysis?studentId=&period=` - 错题分析
  - `GET /report/suggestions?studentId=` - 学习建议

- [ ] **ReportService**
  - `getOverview()` - 获取概览数据
  - `getHomeworkStats()` - 作业统计分析
  - `getMistakeAnalysis()` - 错题分析
  - `generateSuggestions()` - 生成学习建议

### 数据库
- [ ] **learning_reports 表** - 学习报告存储
  - 报告类型（日报/周报/月报）
  - 报告日期
  - 报告数据（JSON）
  - 索引优化

## 🔧 技术实现

### 前端技术
- **图表库** - 使用 F2 或 ECharts for WeChat
- **数据可视化** - 柱状图/折线图/饼图
- **颜色方案** - 科目配色（语文红/数学蓝/英语橙）
- **响应式布局** - 适配不同屏幕

### 后端技术
- **数据统计** - MyBatis-Plus 聚合查询
- **趋势分析** - 日期范围分组统计
- **AI 建议** - 基于规则的模板生成
- **缓存优化** - Redis 缓存报告数据

### 定时任务
- **日报生成** - 每天 23:00 自动生成
- **周报生成** - 每周日 23:00 自动生成
- **月报生成** - 每月最后一天 23:00 自动生成
