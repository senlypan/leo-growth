# 小学生友好体验设计规范

> 👧👦 专为 6-12 岁儿童设计  
> **设计原则:** 简单、有趣、直观、鼓励

---

## 🎨 UI 设计规范

### 1. 字体规范

#### 字号要求
```
标题文字：36-40rpx (18-20px)  - 大而清晰
正文文字：30-32rpx (15-16px)  - 易阅读
辅助文字：26-28rpx (13-14px)  - 不小于 13px
按钮文字：32rpx (16px) 以上    - 方便点击
```

#### 字体选择
- 使用系统默认字体
- 避免花哨的艺术字
- 保证清晰度
- 支持简体中文

### 2. 颜色规范

#### 主色调
- **紫色系** (#667eea → #764ba2) - 学业规划师
- **蓝色系** (#3498db) - 学习报告
- **绿色系** (#2ecc71) - 目标管理/完成状态
- **橙色系** (#f39c12) - 警告/待完成
- **红色系** (#e74c3c) - 语文/错误

#### 色彩使用原则
- ✅ 使用明亮、温暖的色彩
- ✅ 每个功能有专属颜色
- ✅ 状态用颜色区分（绿=完成，橙=进行中）
- ❌ 避免大面积深色
- ❌ 避免过于刺眼的颜色

### 3. 按钮规范

#### 尺寸要求
```
主要按钮：高度 88-96rpx (44-48px)
次要按钮：高度 80-88rpx (40-44px)
图标按钮：最小 48x48rpx (24x24px)
```

#### 设计要求
- ✅ 圆角设计（半径 12-16rpx）
- ✅ 渐变背景（吸引注意力）
- ✅ 文字 + 图标组合
- ✅ 点击反馈（缩放/变色）
- ✅ 足够的间距（防止误触）

### 4. 卡片设计

#### 卡片规范
```
卡片圆角：16rpx (8px)
卡片间距：20-24rpx
卡片内边距：24-32rpx
阴影效果：轻微阴影（友好感）
```

## 🎯 交互设计规范

### 1. 简单直接

#### 操作简化
- ✅ 一页一个主要任务
- ✅ 每屏不超过 3 个操作按钮
- ✅ 重要操作放在底部（拇指易达）
- ✅ 减少输入，多用选择

#### 示例对比
```
❌ 复杂：输入题目 → 选择科目 → 选择日期 → 设置提醒
✅ 简单：选择科目 → 拍照 → 完成
```

### 2. 即时反馈

#### 反馈类型
- **点击反馈** - 按钮缩放/变色（100ms 内）
- **成功反馈** - ✅ + 绿色 + "太棒了！"（1.5 秒）
- **加载反馈** - 动画 + "正在努力..."
- **错误反馈** - ❌ + 橙色 + 鼓励话语

#### 反馈文案
```
✅ 正确：
- "太棒了！"
- "真厉害！"
- "继续加油！"
- "你又进步了！"

⏳ 加载中：
- "正在思考..."
- "马上就好..."
- "稍等一下哦..."

❌ 错误：
- "没关系，再试一次！"
- "慢慢来，你可以的！"
- "这个有点难，我们再看看"
```

### 3. 进度可视化

#### 进度条设计
```
样式：圆角条形
高度：12-16rpx
颜色：渐变绿色（完成感）
动画：平滑过渡（0.3s）
```

#### 完成度展示
- 🌟 星星奖励（每完成一个任务得 1 星）
- 🔥 连续打卡（显示连续天数）
- 📊 进度条（直观看到完成度）
- 🏆 成就徽章（特殊成就）

### 4. 错误预防

#### 防误触设计
- 重要操作二次确认
- 删除操作需要长按
- 返回时自动保存草稿
- 超时自动保存

#### 输入优化
- 能选择就不输入
- 提供默认值
- 输入框足够大
- 数字输入用数字键盘

## 🎮 游戏化设计

### 1. 积分系统

#### 获取积分
```
完成作业：+10 分
全对：+5 分额外奖励
连续打卡：+3 分/天
错题复习：+5 分/题
评估测试：+20 分/次
```

#### 消耗积分
```
兑换虚拟装扮：50-200 分
解锁新主题：100-300 分
兑换实物奖励：500-1000 分（需家长确认）
```

### 2. 成就系统

#### 成就类型
```
📚 学习达人
- 完成 10 次作业
- 连续 7 天打卡
- 作业全对 5 次

🎯 目标达成
- 完成 1 个目标
- 完成 5 个目标
- 完成 10 个目标

📊 进步之星
- 评估分数提升 10 分
- 评估分数提升 20 分
- 评估分数达到 90+

🏃 坚持不懈
- 连续打卡 7 天
- 连续打卡 30 天
- 连续打卡 100 天
```

#### 成就展示
- 成就墙（展示已获得的徽章）
- 进度提示（还差多少获得下一个成就）
- 分享功能（可以分享给家长）

### 3. 鼓励系统

#### 随机鼓励语
```
登录时：
- "今天也要加油哦！"
- "准备好学习了吗？"
- "又是进步的一天！"

完成任务时：
- "太棒了！你又完成了一个任务！"
- "真厉害！继续加油！"
- "哇！你做得太好了！"

连续打卡时：
- "已经坚持 X 天了，真棒！"
- "离目标越来越近了！"
- "你就是最棒的！"
```

## 📱 页面设计规范

### 1. 首页设计

#### 要求
- 欢迎语 + 孩子姓名
- 今日任务一目了然
- 快捷入口（不超过 4 个）
- 积分/成就展示

#### 布局
```
┌─────────────────────────────────┐
│  👋 你好，灏成！                │
│  今天也要加油哦～ 🌟            │
├─────────────────────────────────┤
│  今日任务 (3/5)                 │
│  ████████░░░░░░░░ 60%           │
├─────────────────────────────────┤
│  [📚 作业] [❌ 错题] [📊 报告]  │
├─────────────────────────────────┤
│  🏆 我的成就：12 个徽章          │
└─────────────────────────────────┘
```

### 2. 列表页设计

#### 要求
- 卡片式展示
- 状态清晰标识
- 操作按钮明显
- 支持筛选（简单）

#### 卡片元素
```
┌─────────────────────────────────┐
│  📖 语文           ⏳ 未完成    │
├─────────────────────────────────┤
│  完成练习册第 5 页               │
│  背诵古诗《春晓》               │
├─────────────────────────────────┤
│  ⏰ 截止：今天                  │
│  [✏️ 去完成 →]                  │
└─────────────────────────────────┘
```

### 3. 详情页设计

#### 要求
- 信息层次清晰
- 重点内容突出
- 操作步骤简单
- 返回按钮明显

### 4. 表单页设计

#### 要求
- 表单项不超过 5 个
- 必填项明显标识
- 提供示例/提示
- 提交前确认

## 🎨 动画设计规范

### 1. 转场动画

#### 页面切换
- 进入：从右滑入（0.3s）
- 退出：从右滑出（0.3s）
- 弹窗：从底部弹出（0.3s）

### 2. 反馈动画

#### 成功动画
- ✅ 放大 + 旋转（0.5s）
- 彩带动画（1s）
- 星星闪烁（0.3s × 3）

#### 加载动画
- 旋转圆圈
- 跳动圆点
- 进度条

### 3. 微动画

#### 按钮点击
- 缩放：scale(0.95)
- 时长：100ms
- 缓动：ease-out

#### 卡片悬停
- 轻微上浮：translateY(-4rpx)
- 阴影加深
- 时长：200ms

## 📏 可访问性设计

### 1. 视觉友好

- 对比度足够（文字清晰可读）
- 不使用纯黑纯白
- 色盲友好（不只用颜色区分状态）
- 支持系统字体大小

### 2. 操作友好

- 点击区域足够大（最小 44x44px）
- 支持滑动操作
- 支持返回手势
- 长按操作有提示

### 3. 认知友好

- 使用孩子能懂的语言
- 避免专业术语
- 多用图标辅助理解
- 提供操作引导

## ✅ 检查清单

### 设计检查
- [ ] 字体大小适合儿童阅读
- [ ] 按钮足够大，方便点击
- [ ] 颜色明亮但不刺眼
- [ ] 图标清晰易懂
- [ ] 页面简洁，不拥挤

- [ ] 鼓励语温暖正面
- [ ] 进度可视化

- [ ] 语言简单易懂
- [ ] 没有专业术语
- [ ] 有示例/提示
- [ ] 有完成引导

**设计原则总结:**
1. **简单** - 一页一事，操作简单
2. **有趣** - 游戏化，有奖励
3. **直观** - 一看就懂，一点就会
4. **鼓励** - 正面反馈，建立自信

**目标:** 让 7 岁孩子也能独立使用！🎉
