## 🚀 立即开始

### 1. 数据库初始化
```bash

mysql -u root -p leo_growth < backend/src/main/resources/db/init-full.sql
```

### 2. 启动后端服务
```bash
cd backend
mvn spring-boot:run
```

### 3. 前端测试
1. 打开微信开发者工具
2. 导入 `frontend` 目录
3. 修改 `utils/api.js` 中的 API 地址
4. 编译运行
