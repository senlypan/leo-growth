# 首页优化 - 小学生友好版

> 👋 让首页更吸引孩子，更温暖友好  
> **目标:** 让孩子一打开就喜欢！

---

## 🎨 设计原则

### 1. 温暖欢迎
- 孩子姓名 + 欢迎语
- 鼓励性话语
- 可爱 emoji
- 明亮色彩

### 2. 清晰任务
- 今日任务一目了然
- 进度条可视化
- 完成状态清晰
- 简单操作

### 3. 快速入口
- 常用功能前置
- 大图标 + 文字
- 色彩区分
- 不超过 4 个

### 4. 成就展示
- 积分/徽章展示
- 进步可视化
- 庆祝动画
- 分享功能

## 🎨 视觉设计

### 顶部欢迎区
```css
背景：渐变紫色 (#667eea → #764ba2)
字体：白色，36-40rpx
图标：大 emoji (👋, 🌟)
动画：轻微浮动
```

### 任务进度区
```css
背景：白色卡片
进度条：绿色渐变
完成项：绿色 ✓
未完成：灰色 ⬜
字体：30-32rpx
```

### 快捷入口区
```css
按钮：大图标 (64rpx) + 文字
色彩：每个功能专属色
圆角：16rpx
阴影：轻微
```

### 成就展示区
```css
背景：浅色卡片
数字：大而醒目 (40rpx)
图标：彩色 emoji
标签：小字 (24rpx)
```
