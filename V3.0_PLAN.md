## 🎯 模块目标

帮助学生科学记录体能数据，制定运动计划，培养锻炼习惯。

### 核心功能
1. **体测记录** - 记录身高体重、肺活量、跑步等数据
2. **运动计划** - 制定每日/每周训练计划
3. **训练打卡** - 每日打卡，培养习惯

## 🔧 技术设计

### 数据库设计

#### 体测数据表
```sql
-- 体测记录表
CREATE TABLE physical_records (
    id BIGINT PRIMARY KEY,
    student_id BIGINT NOT NULL,
    record_date DATE NOT NULL,
    height DECIMAL(5,2) COMMENT '身高 cm',
    weight DECIMAL(5,2) COMMENT '体重 kg',
    lung_capacity INT COMMENT '肺活量 ml',
    heart_rate INT COMMENT '静息心率',
    fifty_meter_run DECIMAL(4,2) COMMENT '50 米跑 秒',
    sit_and_reach DECIMAL(5,2) COMMENT '坐位体前屈 cm',
    jump_rope INT COMMENT '跳绳 次/分钟',
    sit_ups INT COMMENT '仰卧起坐 次/分钟',
    standing_jump DECIMAL(5,2) COMMENT '立定跳远 cm',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 运动计划表
```sql
-- 运动计划表
CREATE TABLE exercise_plans (
    id BIGINT PRIMARY KEY,
    student_id BIGINT NOT NULL,
    plan_type VARCHAR(20) NOT NULL COMMENT 'daily-每日，weekly-每周',
    plan_date DATE NOT NULL,
    exercises JSON NOT NULL COMMENT '训练项目',
    completed TINYINT DEFAULT 0,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 训练动作库
CREATE TABLE exercise_items (
    id BIGINT PRIMARY KEY,
    category VARCHAR(50) NOT NULL,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    duration INT COMMENT '时长 分钟',
    difficulty INT COMMENT '难度 1-5',
    video_url VARCHAR(255),
    image_url VARCHAR(255)
);
```

#### 打卡记录表
```sql
-- 训练打卡表
CREATE TABLE training_checkins (
    id BIGINT PRIMARY KEY,
    student_id BIGINT NOT NULL,
    checkin_date DATE NOT NULL,
    plan_id BIGINT,
    duration INT COMMENT '训练时长 分钟',
    calories INT COMMENT '消耗卡路里',
    mood VARCHAR(20) COMMENT '心情',
    note TEXT,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    KEY `idx_student_date` (student_id, checkin_date)
);
```

### API 设计

#### 体测记录 API
```
GET    /physical/records         # 获取体测记录
POST   /physical/records         # 录入体测数据
GET    /physical/trend           # 数据趋势
GET    /physical/standard        # 体测标准对比
```

#### 运动计划 API
```
GET    /exercise/plans           # 获取训练计划
POST   /exercise/plans           # 创建训练计划
PUT    /exercise/plans/{id}      # 更新计划
POST   /exercise/plans/{id}/complete  # 完成计划
GET    /exercise/items           # 获取动作库
```

#### 训练打卡 API
```
GET    /training/checkins        # 获取打卡记录
POST   /training/checkins        # 打卡
GET    /training/stats           # 打卡统计
GET    /training/calendar        # 打卡日历
```
